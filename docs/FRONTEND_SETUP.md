# FiscX Frontend

Application web React pour commerçants, banques et régulateurs.

## Structure

```
src/
├── pages/          # Pages (Login, Dashboard, etc.)
├── components/     # Composants réutilisables
├── stores/         # État Zustand
├── api/            # Client API Axios
├── utils/          # Utilitaires
├── index.css       # Styles Tailwind
└── App.jsx         # Application principale
```

## Installation

```bash
npm install
npm run dev
```

L'app démarre sur `http://localhost:5173`

## Build

```bash
npm run build
```

Production: `npm run preview`
